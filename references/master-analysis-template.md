# 棠溪 · 旧统一分析模板 v3.0（已废弃）

本文件已废弃，不再作为权威模板来源。

权威模板指向：`references/master-template-v68.md`

废弃原因：

① 与 `master-template-v68.md` 同时声明权威来源，造成模板路由冲突。
② 含旧评分换算，与当前锁定的 13分制不一致。
③ 旧文案的模型数量与实际六个执行模型不一致，造成模型边界冲突。
④ XAU 数据规则仍把 Yahoo 期货当现货源，不符合现货/basis 分离的新规则。
⑤ 文档内使用表格、旧操作段样式，与当前纯纵向输出规范不一致。

迁移规则：

① 所有分析卡、监控卡、auto_card、技能引用，均以 `references/master-template-v68.md` 为准。
② 本文件仅保留为历史记录，不得直接用于出卡。
③ 如需查看旧版本差异，通过 git 历史检索，不在运行管线中引用本文件。

当前权威模板核心：

① 头部10段。
② 正文五段：环境、结构、博弈、操作、风控。
③ 六类固定执行模型：VWAP反抽、VAH回收、VAL回收、POC拒绝、扫流动性回收、突破接受。
④ 13分制评分。
⑤ 双预案A/B。
⑥ R:R≥1:2硬底线。
⑦ setup_id/model_id/entry_tag/exit_tag 机器字段。
⑧ 数据、事件、R:R、日损、连亏、执行距离风控闸门。
