## 附：六类固定执行模型触发清单

### VWAP反抽

```
入场前：① 4h/1h同向不冲突 ② 价格距VWAP≤0.5ATR ③ 前段趋势无明显衰竭
触发：① 触碰VWAP后拒绝 ② 5m/15m收线不重新接受VWAP ③ Taker/CVD不冲突
确认：① 顺向K线重启 ② 反向量能衰竭 ③ 回踩/反抽不破确认位
失效：① 重新站回VWAP并连续接受 ② CVD持续背离 ③ 高周期关键位被反向收复
```

### VAH/VAL回收

```
入场前：① 价格打出VAH上方或VAL下方 ② 未形成接受（假突破嫌疑）③ CVD/Taker未继续放大
触发：① 价格回到价值区内 ② 5m/15m收线确认回收 ③ 扫出一侧流动性后无法延续
确认：① 不再跌回/涨回价值区外 ② CVD背离或主动成交转向 ③ 目标至少POC或价值区另一侧
失效：① 重新突破VAH/VAL并接受 ② 回踩边界不破后继续顺势 ③ 回收后量能不足且横住
```

### POC拒绝

```
入场前：① 价格回到POC或高成交节点 ② 不是强趋势单边接受 ③ POC附近有拒绝空间
触发：① 触碰POC后失败 ② 5m/15m出现拒绝K线或反向主动成交 ③ POC未被连续收复
确认：① 拒绝后离开POC区域 ② 成交量支持反向 ③ CVD/Taker不反向冲突
失效：① POC被收复并接受 ② 回踩POC不破并继续原方向 ③ 围绕POC反复横盘
```

### 扫流动性回收

```
入场前：① 接近前高/前低/亚洲高低/纽高纽低/止损池 ② 高周期无强烈反向压制 ③ 扫出后观察主动成交是否衰竭
触发：① 刺破关键高/低 ② 刺破后快速回收 ③ 5m/15m收回关键位内侧
确认：① CVD背离或Taker追单失败 + 流动性扫荡确认 + Displacement ② 回收后不二次跌破/突破扫点 ③ 出现反向结构位突破
失效：① 扫后继续放量延续 ② 回收失败且在扫点外接受 ③ 高周期同步破位
```

### 突破接受

```
入场前：① 接近VAH/VAL/VWAP/前高前低 ② 高周期支持突破方向 ③ 突破前无明显衰竭
触发：① 价格突破关键位 ② 5m/15m收线站稳关键位外侧 ③ 回踩关键位不破
确认：① CVD/Taker顺向确认（无背离） + Displacement ② OI增加但不过度拥挤 ③ 回踩后二次启动
失效：① 突破后快速回落价值区内 ② 回踩跌破/上破失败 ③ CVD背离且Taker追单失败
```
