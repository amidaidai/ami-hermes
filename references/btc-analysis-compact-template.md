# BTC分析精简模板 v1（2026-06-22 社区审计优化版）

## 目标
30秒读完决策核心，去掉冗余分析，聚焦行动。

## 模板

```
**① 品种 · 方向**
`BTCUSDT.P:BINANCE` · HTF空 · 顺空优先

**② 关键位**
阻 `64,208`VWAP / `64,566`POC / `64,672`VAH
支 `63,947`VAL / `63,312`DO

**③ 现价 · 数据 · 量能**
`63,934` · 数据B·CVD顺空 · 量70极缩

**④ 入场条件**（满足才进）
空：放量破VAL`63,947`或缩量持续在VAL下+指标转A空
多：放量站回VAL+VWAP → 假破确认
否：缩量期不动

**⑤ 核对清单**
□ HTF方向一致？
□ 放量确认？
□ R:R≥1:2？
□ 无重大事件？
□ 情绪不极端反向？

**⑥ 预案**
A 空 → 入场`63,800` 止损`64,208` 止盈`63,312`(DO) 1:2.5
B 多 → 入场`63,947` 止损`63,800` 止盈`64,208`(VWAP) 1:2.5
```

## 与旧模板的差异

| 旧 | 新 | 理由 |
|----|----|------|
| 10段头部 | 6段精简 | 去掉重复的评分/模型/仓位计算 |
| 5段正文 | ⑥预案合一段 | 环境和博弈合并到方向+关键位 |
| 4周期逐层写 | 只看方向层和执行层 | 1h/5m中间层日常噪音 |
| 独立因子评分 | 引用指标等级+核对清单 | 不再重复SVP指标的工作 |
| 双路A/B详细展开 | A/B缩进行 | 操作段在精不在长 |
