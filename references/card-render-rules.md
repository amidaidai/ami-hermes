# 棠溪分析卡 · 渲染规则 v6.9.4

定位：承接主模板拆出的模型、评分、数据定级、社区共识、本地闭环规则。主模板只负责出卡正文格式。

## 社区对标依据

① Freqtrade：信号计算向量化，入场、止损、仓位、杠杆、订单超时用回调分层；重计算不要放在高频回调。
② QuantConnect：回测与实盘必须建模手续费、滑点、成交、保证金、期权定价、容量限制。
③ NautilusTrader：数据、策略、执行、组合、回测引擎分层；多品种数据先统一排序与时间口径。
④ ICT/SMC/订单流社区：Liquidity Sweep 必须锚定关键位，并用 CVD背离/吸收 + 强位移确认；孤立订单流不计分。
⑤ 黄金外汇社区：London/NY Kill Zone时段是过滤器，常见序列为 Asia筑范围 → London扫荡 → NY确认。
⑥ 期权社区：方向判断之外必须写 Delta、Gamma、Theta、Vega、IV、到期日与最大亏损。

## 六类执行模型

### VWAP反抽
```
入场前：4h/1h不冲突 — 价格距VWAP≤0.5ATR — 前段趋势未衰竭
触发：触碰VWAP拒绝 — 5m/15m不重新接受VWAP — Taker/CVD不冲突
确认：顺向K线重启 — 反向量能衰竭 — 回踩/反抽不破确认位
失效：重新接受VWAP — CVD持续背离 — 高周期关键位反向收复
```

### VAH/VAL回收
```
入场前：价格打出价值区外 — 未形成接受 — CVD/Taker未继续放大
触发：价格回到价值区内 — 5m/15m收线确认 — 扫一侧流动性后无法延续
确认：不再跌回/涨回价值区外 — CVD背离或主动成交转向 — 目标至少POC或价值区另一侧
失效：重新突破并接受 — 回踩边界不破后继续顺势 — 回收后量能不足横住
```

### POC拒绝
```
入场前：价格回到POC/HVN — 不是强趋势单边接受 — 附近有拒绝空间
触发：触碰POC失败 — 5m/15m出现拒绝K线或反向主动成交 — POC未被连续收复
确认：拒绝后离开POC区域 — 成交量支持反向 — CVD/Taker不反向冲突
失效：POC被收复并接受 — 回踩POC不破继续原方向 — 围绕POC反复横盘
```

### 扫流动性回收
```
入场前：接近前高/前低/亚洲高低/止损池 — 高周期无强反向压制 — 等主动成交衰竭
触发：刺破关键高低 — 快速回收 — 5m/15m收回关键位内侧
确认：CVD背离或吸收 — 强位移反向确认 — 反向结构位突破
失效：扫后继续放量延续 — 回收失败并在扫点外接受 — 高周期同步破位
```

### 突破接受
```
入场前：接近VAH/VAL/VWAP/前高低 — 高周期支持突破方向 — 突破前无明显衰竭
触发：突破关键位 — 5m/15m收线站稳外侧 — 回踩关键位不破
确认：CVD/Taker顺向且无背离 — OI增加但不过度拥挤 — 回踩后二次启动
失效：突破后快速回落价值区内 — 回踩失败 — CVD背离且Taker追单失败
```

## 评分与数据定级

13分制：
① 结构 0-3，权重最高 — 趋势、价值区、关键位是否清晰。
② 周期 -2到2 — 高周期是否支持低周期方向。
③ 订单流/量价 0-2 — CVD、Taker、吸收、背离、成交量是否确认。
④ 衍生/宏观/希腊值 0-2 — 按资产类选择有效因子。
⑤ 催化 0-1 — 新闻、财报、宏观日历需至少两源确认。
⑥ 风控 0-2，权重最高 — R:R、止损、日限、执行距离。
⑦ 情绪 -1到1 — 极端拥挤降权，恐慌小幅加权。

状态判定：
① 10-13：A做多 或 A做空。
② 6-9：B等待。
③ 0-5：X禁做。
④ 数据 C 级：最高 B等待。
⑤ CVD/量价 C 级：订单流分降权，仓位上限半仓。
⑥ 4h 与 1h 同时冲突：X禁做。
⑦ R:R 低于 1:2：X禁做。
⑧ 日损达线、连亏3笔、重大事件窗口：X禁做。

数据定级：
① A：价格多源一致、关键衍生/宏观/量价可用、数据≤15分钟。
② B：价格两源一致，部分因子缺失但不影响核心判断。
③ C：单源、价差异常、关键订单流/宏观/希腊值缺失，最高 B等待。

CVD/量价定级：
① A：逐笔成交或可信订单流源。
② B：Taker买卖量或可靠成交量代理。
③ C：K线估算，只能辅助，不能单独触发。
