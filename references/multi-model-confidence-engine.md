## 置信公式（每个模型独立计算）

```text
模型置信 = 条件满足率 × 数据质量系数 × 方向一致性加分

其中：
- 条件满足率 = 满足的入场条件数 / 总条件数（0-1）
- 数据质量系数：A=1.0, B=0.8, C=0.5
- 方向一致性加分：高周期支持+0.2，衍生品支持+0.2，关联支持+0.1
  
模型置信 ≥ 0.6 → 强信号
模型置信 0.4-0.6 → 中等信号
模型置信 0.2-0.4 → 弱信号
模型置信 < 0.2 → 噪音，不计入合并
```

## 合并算法

```text
1. 所有模型独立跑一遍，输出 (方向, 置信)
2. 按方向分组：做多组 / 做空组
3. 每组计算：
   综合置信 = Σ(模型置信²) / Σ(模型置信)
   （加权平均，高置信模型权重大）
   信号数量 = 该方向模型个数
4. 判定：
   做多综合置信 − 做空综合置信 > 0.2 → 偏多
   做空综合置信 − 做多综合置信 > 0.2 → 偏空
   差距 ≤ 0.2 → 方向不明 / 震荡
5. 全局置信 = max(做多综合, 做空综合)
   全局置信 < 0.4 → 不交易
   全局置信 0.4-0.6 → 轻仓
   全局置信 0.6-0.8 → 常规
   全局置信 > 0.8 → 高置信
```

## 品种模板变量

不同品种使用不同的模型子集：

**BTC/ETH（全量）：**
- 五类固定 + EMA趋势 + 费率极端 + 多空拥挤 + Taker背离 + OI背离 + M_VWAP磁吸 + 关联套利（13个）

**山寨：**
- 五类固定（减半权重）+ EMA趋势 + BTC相关性（山寨专用模型：BTC稳定时山寨突破/补涨）

**XAU：**
- 五类固定 + EMA趋势 + DXY背离 + 美债背离 + 时段模型（London/NY开盘效应）+ 地缘催化

**外汇：**
- EMA趋势 + 套息交易（利率差）+ DXY关联 + 时段模型 + 宏观催化

## 冲突信号处理

当多个模型输出矛盾时：

```text
1. 高周期信号权重 ×1.5
2. 衍生品信号（费率/多空/OI）权重 ×1.2
3. 结构信号（五类固定）权重 ×1.0
4. 情绪/关联信号权重 ×0.5

加权后再合并 → 高周期判断优先于低周期
```
